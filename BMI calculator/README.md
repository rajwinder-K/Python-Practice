# BMI Calculator

A simple Python program to calculate **Body Mass Index (BMI)** based on user input for weight and height.

# Features
- Prompts the user to enter their **weight (kg)** and **height (m)**.
  
- Calculates BMI using the formula:

  
\[
  BMI = \frac{weight}{height^2}
  \]


- Displays the entered weight, height, and the calculated BMI (rounded to 2 decimal places).


# How to Run

1. Make sure you have **Python 3** installed.
2. Save the script as `bmi_calculator.py`.
3. Run the program in your terminal:

   ```bash
   python bmi_calculator.py

# Example Output
```
Weight(kg): 70
Height(m): 1.75
Weight : 70.0
Height : 1.75
BMI    : 22.86
```
# Notes
BMI is a general indicator of body fat but does not account for muscle mass, age, or other health factors.

Use this program for educational purposes only. For medical advice, consult a professional.
