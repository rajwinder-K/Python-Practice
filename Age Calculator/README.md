# Age Calculator

A beginner Python project that calculates your age in **years** based on your birth year.

## Features

- Asks for your birth year
- Uses Python's `datetime` module to get the current year
- Calculates age in years
- Handles invalid input and prevents negative ages
## Example 

Enter your birth year: 2000
Birth Year    : 2000
Current Year  : 2026
Your age is   : 26
