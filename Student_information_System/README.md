# Student Card Program
# Overview
This is a simple Python program that asks the user for student details (Name, Roll Number, Age, Branch, Semester, CGPA) and prints them in a clean Student Card format.
# Features
- Collects student details interactively.

- Converts inputs into correct data types (int, float, str).
 
- Displays a formatted student card using f-strings.

- Great for beginners practicing Python basics.
  
# Example Output
```text
Name: Komal
Roll Number: 202
Age: 21
Branch: IT
Semester: 6
CGPA: 9.1

------ Student Card ------
Name         : Komal
roll_number  : 202
age          : 21
branch       : IT
semester     : 6
cgpa         : 9.1
```
# Learning Goals
This project helps practice:

- Python input handling

- Data type conversion (int, float, str)

- String formatting with f-strings
