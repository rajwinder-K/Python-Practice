# Currency Converter

A simple Python program to convert between Indian Rupees (INR) and US Dollars (USD).

# Features
- Convert INR → USD

- Convert USD → INR

- Easy-to-use menu-driven interface

- Uses a fixed exchange rate (1 USD = 86 INR)

# How It Works

1. Run the program.

2. Choose an option:

   - 1 for INR to USD

   - 2 for USD to INR

3. Enter the amount.

4. Get the converted value instantly.

# Example 1.
```text
1. INR to USD
2. USD to INR
Enter your choice: 1
Enter Amount in INR: 860
860.00 rupee = $10.00
```
# Example 2.
```text
1. INR to USD
2. USD to INR
Enter your choice: 2
Enter Amount in USD: 5
$5.00 = 430.00 rupee
```
