## Login System

This is a beginner practice project in Python.  
It shows how to:
- Ask for a username and password
- Hide password input using `getpass`
- Check login details with `if` / `else`
- Support multiple users with a dictionary
- 
## Notes
- Usernames are **case-sensitive**.
- Passwords are stored in plain text (for practice only).
- This is not a production system — just a learning exercise.
