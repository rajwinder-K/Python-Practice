# Number Guessing Game

This is a small Python project I made to practice loops, conditionals, and random numbers.

## How it works
- The game asks you to choose a difficulty level:
  - Easy (1–50)
  - Medium (1–100)
  - Hard (1–1000)
- A random number is picked in that range.
- You keep guessing until you find the correct number.
- The program tells you if your guess is too high or too low.
- It also gives a hint if you are very close.
- At the end, you can choose to play again.

## Example

Choose Difficulty Level:
Easy (1–50)
Medium (1–100)
Hard (1–1000)
Enter your choice (1/2/3): 2
You chose Medium! Guess the number between 1 and 100.
Enter a number: 50

Too low, try again!
